Архив сайта Strike Zone Bowling Club, исправленный для GitHub Pages.

Как загрузить в репозиторий sitee:
1. Распакуйте архив.
2. Загрузите все файлы и папки из архива прямо в корень репозитория sitee, не внутрь дополнительной папки.
3. В GitHub Pages выберите Source: Deploy from a branch, Branch: main, Folder: / (root).
4. Сайт будет открываться по адресу вида https://имя_пользователя.github.io/sitee/.

Что исправлено:
- страницы и ресурсы лежат в корне архива, без лишней вложенной папки bowling_site;
- пути к CSS, JS и изображениям сделаны относительными через ./assets/...;
- добавлен файл .nojekyll, чтобы GitHub Pages не обрабатывал сайт через Jekyll.

Фотографии и изображения не изменялись.
